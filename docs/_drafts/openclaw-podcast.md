Eve: 为什么很多人要请人安装 OpenClaw？OpenClaw 很难安装吗?

Devon: 对，OpenClaw 并没有一个图形化的安装界面,其实就算是对程序员来说，OpenClaw都很难安装. 首先就是慢，因为它的安装包在 GitHub 或者 npm 上面，其实对程序员来讲，可能多数人知道 GitHub，但不是所有人知道 NPM。如果安装包在github和npm 上的话，那安装包下载就比较慢，所以涉及到要设置镜像。这些东西看起来就太技术了。

另外 OpenClaw 对运行环境有一些依赖。最主要的依赖就是对 NodeJS版本 的依赖。如果你不是前端程序员，哪怕是程序员，也不一定搞得清楚如何安装它，比如刚好需要的那个 node.js 版本。

一旦这个过程出错，非程序员就很难去恢复它. 

而且，安装只是开始，安装完成后还需要进行设置。对于新手来说，主要不懂这些设置是干什么用的。尤其是这些设置还依赖一些外部的 API Key，可能都不知道这些 Key 要上哪里去，也不知道要满足什么样的条件去申请。而且，很多 API Key 都是使用国外服务的，那你可能在上网的时候就会遇到困难。如果你不配置，又不清楚到底有哪些功能，那就变得不可用。

Aaron: 对，我感觉最主要的配置有三块：首先是要配置模型，其次是要配置搜索引擎，最后是要配置通信的渠道。在这三个配置中，作为普通用户最容易理解的就是配置通信渠道，就是要给他绑上微信或者 QQ。 为什么要配置模型呢？

Devon: 对于大模型来说，OpenClaw 的大脑只有配置了大模型，OpenClaw 才能够思考。它能够读懂用户发送的消息，懂得什么时候去调用工具，以及如何搜索、整理信息，最后回复给用户. OpenClaw 在这里最大的问题是，它一上来就提供了很多模型给用户选择。但是这些模型其实普通用户一个都不认识，而且手上一个也没有，所以根本就不知道该怎么配.

Aaron: 对，我刚开始用的时候确实也遇到这个问题。就是它的模型列表使用太技术化了，其实你看不懂到底是什么意思。比方说我可能只知道gemini，只知道qwen。然后他假设你知道什么是 API key，以及 API key 需要到什么地方申请. 你可能真的不知道 Gemini 的 API 要去哪里申请，其实要搞定这个还是很难的。虽然我知道 Qwen 的模型很厉害，但我不知道它的 API 要去百炼的平台去申请。

另外它有一个比较坑的点：它其实默认是让你用 OpenRouter 的。这跟我们的方案也是一样，但它使用的是 OpenRouter 的 Auto 接口聚合。这个接口里面包含一些付费模型，所以你有可能刚开始几句话还能聊下来，但很快就会因为欠费导致不能用。这对新人来讲其实非常不友好。

Muse: 这是我们产品的一个很大改进点：我们提供给大家的是一个虚拟机镜像。只要你机器上有虚拟机软件，把这个镜像装载进来。前面说的那些复杂的安装完全不需要. 它就像一台电脑，在初次启动时会有一个欢迎界面,这个欢迎界面的第一步，就是让你去设置一个 OpenRouter 的 API key.

在这个页面里面，我们给出了 OpenRouter 官网的地址，所以你点一下就可以进入，到创建 API 的地方。在 OpenRouter 上面其实是创建 API Key 而不是申请 API Key，因为创建 API Key 是免费的，没有任何条件的。所以你只要会上网，会注册账号就行了。

当你创建了 API key 之后，把它填回到我们给出的欢迎界面，它就会自动帮你去查询哪些大模型是免费使用的。通常它会查询出来 7、8 个免费的大模型。最先返回的是最新的大模型，一般来讲也是能力最强的大模型。

Aaron: 这解决了第一个配置问题。关于第二个配置问题，也就是搜索引擎这一块儿, 我们是怎么处理的? 首先，Devon 介绍一下，为什么 OpenClaw 要配置搜索引擎?

Devon: 大模型训练的时候，都是使用截止到某一天之前的知识进行训练的，它对发布之后的知识是一无所知的. 比方说，几乎没有大模型知道什么叫 OpenClaw, 在这个时间点发布的大模型，它们训练时用的语料，一般都是 2026 年元旦之前的. 所以大模型相当于,第一，它拥有世界的知识，第二，最重要的是，它了解知识与知识之间的关联，具备逻辑推理的能力。但是如果你问他对最新的热点事件怎么看，那他就要上网去搜索这个热点问题，拿到一些资料，然后再进行分析推理，形成观点。所以对 OpenClaw 来讲，搜索引擎是非常重要的。

Muse: 在 OpenClaw 初始化的时候，它会显示一个搜索引擎的列表，上面列了好多搜索引擎，但是这些搜索引擎都是面向 AI 的，所以你平时其实很可能都没有听说过，所以你选择的依据基本上就是免费或者付费。我们的做法是，默认地把免费的搜索引擎，也就是 DuckDuckGo 给配置了，然后我们会安装浏览器，后面你就可以通过 OpenClaw 让它通过。浏览器自己去访问一些重要的网站，获得最新的资讯。这样你就可以不用付费的搜索引擎了。

Eve: 这也是我们降成本一个很重要的方法. 因为即使是免费的搜索引擎，一个月最多也能够给到 1000 次免费搜索，完全不够用.

Muse: 所以第二个配置就这样解决了。绑定微信这一步很简单也很直观, 我们会在界面上显示一个二维码，大家直接扫码就可以了。但是添加 QQ 这里，也会涉及一些专业术语，Devon 你给大家解释下。

Devon: 添加 QQ 不是通过扫码完成的，需要我们提供 appId 和 appSecret。这一步其实也不难，关键是你要知道去哪里申请 API id 和 APP Secret

在界面上我们已经给出了 QQ 官网上的一个链接，当你点击它的时候，就会直接跳转到申请的页面

这里就会显示一个 app Id 和 app Secret，同时还会显示一个添加机器人的二维码。你需要用自己的 QQ 扫描这个二维码，就能够添加成功这个机器人，然后把 APP ID 和 app secret 填回到我们的界面上，点击下一步，这个绑定就能成功.

绑定成功后，你就会在微信或 QQ 上收到一条消息，然后你就可以开始使用小龙虾了.
